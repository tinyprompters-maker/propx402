## What is PropX402?

PropX402 aggregates **14 government and public data sources** into a single structured JSON response — delivered in 3–8 seconds. Payments happen natively via the [x402 protocol](/x402-protocol) on Base mainnet. No accounts, no subscriptions, no dashboards.

## Who is it for?

- **AI real estate agents** — structured property data without scraping or manual lookups
- **Investor tools** — run deal analysis at scale across any US address
- **Autonomous agents** — ElizaOS, GOAT, LangChain, or any x402-enabled framework
- **MCP clients** — Claude Desktop, Cursor, or any MCP-compatible tool
